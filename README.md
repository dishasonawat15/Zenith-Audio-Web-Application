# Zenith-Audio-Web-Application
Zenith Audio is a design-focused frontend project built to showcase modern web design principles. Rather than relying on heavy JavaScript frameworks, this project demonstrates a strong command of raw HTML5 and CSS3 to create an engaging, responsive, and visually appealing e-commerce landing page and authentication flow.

## ✨ Key UI/UX Features
* **Modern Glassmorphism:** Utilizes CSS `backdrop-filter` and semi-transparent backgrounds to create a premium, frosted-glass aesthetic on the login and feature cards.
* **Responsive Layouts:** Built with CSS Flexbox and media queries to ensure a seamless experience across desktop and mobile devices.
* **Interactive Authentication UI:** Features a smooth, lightweight JavaScript toggle between Login and Sign Up states without requiring a page reload.
* **Custom Theming:** Utilizes CSS Variables (`:root`) for consistent color theming (Mint Green/Subtle Whites) and easy global style updates.
* **Polished Typography:** Integrates Google Fonts ('Anton' for bold branding and 'Poppins' for clean readability).

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 (Flexbox, Variables, Animations, Glassmorphism)
* **Interactivity:** Vanilla JavaScript (DOM manipulation for auth toggling)
